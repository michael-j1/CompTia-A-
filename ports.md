| Port | Protocol | Service / Protocol Name | Description |
| :--- | :--- | :--- | :--- |
| **20** | TCP | FTP (Data Transfer) | Used for transferring actual file data. |
| **21** | TCP | FTP (Control) | Used for passing FTP commands and routing. |
| **22** | TCP/UDP | SSH / SFTP | Secure Shell for secure remote login and file transfers. |
| **23** | TCP | Telnet | Unencrypted text communications (legacy remote login). |
| **25** | TCP | SMTP | Simple Mail Transfer Protocol (routing email between servers). |
| **53** | TCP/UDP | DNS | Domain Name System (translates domain names to IP addresses). |
| **67 / 68**| UDP | DHCP | Dynamic Host Configuration Protocol (assigns IPs dynamically). |
| **80** | TCP | HTTP | Hypertext Transfer Protocol (unencrypted web traffic). |
| **110**| TCP | POP3 | Post Office Protocol v3 (retrieving emails from a server). |
| **123**| UDP | NTP | Network Time Protocol (clock synchronization). |
| **143**| TCP | IMAP | Internet Message Access Protocol (retrieving emails with sync). |
| **161**| TCP/UDP | SNMP | Simple Network Management Protocol (monitoring network devices). |
| **443**| TCP | HTTPS | HTTP Secure (encrypted web traffic via SSL/TLS). |
| **445**| TCP | SMB | Server Message Block (file sharing on Windows networks). |
| **587**| TCP | SMTP (Submission)| Modern encrypted email submission port. |
| **3389**| TCP/UDP | RDP | Remote Desktop Protocol (Windows remote desktop connection). |
